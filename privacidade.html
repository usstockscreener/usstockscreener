Política de Privacidade

Última atualização: junho de 2025

1. Informações que coletamos

O US Stock Screener não coleta dados pessoais dos usuários. Não exigimos cadastro, login ou qualquer informação pessoal para usar o site.

2. Cookies e tecnologias de rastreamento

Podemos utilizar cookies de terceiros para fins de publicidade (Google AdSense) e análise de tráfego (Google Analytics). Esses serviços podem coletar dados de navegação de forma anônima, como páginas visitadas, tempo de permanência e localização geográfica aproximada.

Você pode desativar cookies nas configurações do seu navegador a qualquer momento.

3. Publicidade

Este site pode exibir anúncios através do Google AdSense e outras redes de publicidade. Esses serviços podem usar cookies para exibir anúncios relevantes com base em suas visitas anteriores a este e outros sites.

Para mais informações sobre como o Google usa dados, acesse: policies.google.com/technologies/ads

4. Dados de terceiros

Os dados financeiros exibidos no site são obtidos via API do Finnhub (finnhub.io). Consulte a política de privacidade do Finnhub para entender como eles tratam os dados.

5. Segurança

Não armazenamos nenhuma informação pessoal dos usuários em nossos servidores. Todo o processamento de dados financeiros acontece diretamente no navegador do usuário.

6. Alterações nesta política

Podemos atualizar esta política periodicamente. Recomendamos verificar esta página ocasionalmente para se manter informado sobre eventuais mudanças.
